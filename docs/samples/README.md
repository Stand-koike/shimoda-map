# サンプルデータ

## `spreadsheet-headers.csv`

1 行目だけの列ヘッダーです。新規スプレッドシートに貼り付けて列位置を `CONFIG.COLS` と一致させます。

## `demo-spots.csv`

**デモ用の店舗データ約 10 件**です（外浦海岸周辺を想定した緯度経度・ダミー電話番号）。

### スプレッドシートへ転記する手順

1. Google スプレッドシートで、**既存シートの列数・列順**が [SPREADSHEET.md](../SPREADSHEET.md) と一致していることを確認する。
2. `demo-spots.csv` をテキストエディタで開き、**2 行目以降（データ行）だけ**をコピーする。  
   - ヘッダー行は既にスプレッドシートにある場合はコピーしない。
3. スプレッドシートの **データを貼り付けたい行の先頭セル**を選択し、`編集` → `貼り付け` → `貼り付けのみテキストを結合`（または通常貼り付け）で貼る。
4. 列がずれる場合は、**列を挿入**して [SPREADSHEET.md](../SPREADSHEET.md) のインデックスと揃える。

### 画像 URL について

- デモでは **picsum.photos** のプレースホルダ画像を使用しています（そのままピン・カードに表示されます）。
- 本番では **AppSheet の画像列**や **Google Drive** の URL を `image_url` に入れてください。Drive の共有リンクは [index.html](../../map/index.html) 内の `normalizeImageUrl` で表示用 URL に変換されます。

### 注意

- 電話番号・住所は**架空**です。公開前に差し替えてください。
- 緯度経度はイラストマップの表示範囲付近の目安です。必要に応じて [CONFIG.MAP_IMAGE](../../map/index.html) と照らして調整してください。
