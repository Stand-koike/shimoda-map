# 要件定義（外浦海岸マップ Web）

**文書バージョン**: 1.0  
**対象**: ブラウザ上でイラストマップ上に店舗等をピン表示するシングルページアプリ（[`map/index.html`](../map/index.html)）

---

## 1. 背景・目的

- **背景**: 観光協会ウェブサイト内で、外浦海岸エリアの地域・店舗情報を分かりやすく提示する。
- **目的**:
  - 観光客向けに地域情報の可視化とリッチな UI で没入感を高める（体験品質の参考: ディズニーリゾート公式アプリ）。
  - 地元住民の「再発見」にも寄与する。
  - **更新の容易さ**と**情報の鮮度**で他のデジタルマップとの差別化。
- **将来（非必須）**: ルート提案、駐車場混雑情報などの拡張。

---

## 2. 利用想定・環境

| 項目 | 内容 |
|------|------|
| 配信 | 観光協会 HP **内への埋め込み**（iframe）。左遷で**専用タブ／専用ページ**にもできる URL 構成。 |
| 主ユーザー | 観光客。副次として地元住民。 |
| オフライン | 圏外は強く想定しない。初回・実行時の**軽量性**は最低限確保。 |
| 言語 | 日本語・英語（ひとまず）。 |
| 規模 | 店舗数 **約 20** 件程度を想定。 |

---

## 3. データ・運用

| 項目 | 内容 |
|------|------|
| マスタ | **Google スプレッドシート**（マスタ管理者は **1 名**）。 |
| 初期登録 | スプレッドシートへの**一括登録**。 |
| 店舗更新 | 各店舗は **AppSheet** から、**認証により自分の行のみ**編集・更新（基本運用）。 |
| 住所・連絡先 | **必ず表示**（詳細モーダル）。 |
| クーポン | ひとまず**画面提示**で十分。 |
| 位置精度 | 実測との誤差は **半径約 3 m 程度まで許容**。 |

詳細な列定義は [SPREADSHEET.md](./SPREADSHEET.md) を正とする。

---

## 4. 機能要件（概要）

- イラストマップ（`CONFIG.MAP_IMAGE`、既定は高解像度 `300.png` 等）を主表示とし、空白地図スタイルで周囲の一般地図を主表示にしない。
- スプレッドシート由来の緯度経度でピン表示、カテゴリ・タグ絞り込み、カード一覧、詳細モーダル（複数画像スライダー等）、ニュースティッカー、言語切替、現在地（任意）、レイヤー ON/OFF、データの定期再取得と URL による強制更新。

---

## 5. 非機能要件（ドラフト）

| 項目 | 内容 |
|------|------|
| 軽量性 | アセット・ポーリング間隔・再描画の抑制に配慮。 |
| セキュリティ | 公開スプレッドシートに載せる情報の範囲に注意。Mapbox トークンはクライアントに含まれる前提（運用でローテーション等を検討）。 |
| プライバシー | 位置情報利用時はブラウザの許可が必要。 |

## 7. 将来構想（ロードマップ）

以下はフェーズ2以降で検討する機能・活用案です。スプレッドシート・AppSheet・レイヤー管理の基盤を活かして段階的に拡張できます。

### フェーズ2（近期）

| 機能 | 概要 | 主な技術要素 |
|------|------|-------------|
| ❤️ いいねカウンター | ハートボタンで GAS 経由スプレッドシートの数値を +1。人気スポットを可視化 | GAS Webhook + 書き込み API |
| 🔍 タグ詳細化 | `#駐車場あり` `#Wi-Fi` `#電源` 等を AppSheet 選択式でマスター管理 | 実装済みタグ機能の運用整備 |

### フェーズ3（中期）

| 機能 | 概要 | 主な技術要素 |
|------|------|-------------|
| 🎯 デジタルスタンプラリー | 特定スポットのカードを GPS 判定で開くとスタンプ取得。3つでクーポン等の特典 | GPS判定 + LocalStorage/AppSheet |
| 🗺️ イベントリアルタイムマップ | 祭り・パレードの現在地・屋台を期間限定登録。AppSheet でリアルタイム更新 | 既存レイヤー + イベント用シート |
| 🌊 海況・釣りマップ | ダイビングショップ等が「本日の透明度・釣果」を更新。毎朝チェックされるアプリへ | AppSheet 更新 + 専用カテゴリ |

### フェーズ4（長期）

| 機能 | 概要 | 主な技術要素 |
|------|------|-------------|
| 🏡 暮らしマップ | 地元スーパー・コインランドリー・コワーキング等。移住・ワーケーション層向け | 別シート or レイヤー追加 |
| 🆘 防災・避難所モード（フェーズフリー） | スプレッドシート切替で津波避難ビル・高台ルートを表示。観光客の安全確保 | レイヤー切替 + 別シート |
| 🏆 周遊ルート提案 | 徒歩・車の「モデルコース」をルートレイヤーとして表示 | GeoJSON ルートデータ + LayerManager |
